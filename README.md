# Experience OS v3

A mobile-first PWA for tracking lived inputs through four domains:

1. Relationships
2. Growth
3. Contribution
4. Achievement

## Design principle

Optimize for behavior, not completeness.

## Migration

Legacy Experience OS v2 domains are automatically mapped:
- Community → Relationships
- Social → Relationships
- Service → Contribution
- Culture → Growth
- Exploration → Growth
- Achievement → Achievement

## Features

- Dashboard-first layout
- Low-friction capture
- Multi-tag experiences
- Four-domain model
- 7-day experience graph
- 24-hour timestamps
- Local-only storage
- JSON export/import
- Distinct Experience OS journey/path icon set
