# Experiences OS v2

A Foundation OS-style PWA for tracking lived inputs: Community, Social, Service, Culture, Exploration, and Achievement.

## Features
- Dashboard-first layout
- Low-friction capture
- Multi-tag experiences
- Separate scores by pillar
- 7-day experience graph
- 24-hour timestamps
- Local-only storage
- JSON export/import
- Dark mode

## Scoring
Each pillar uses directional visibility scoring:
- 0 entries = 0
- 1 entry = 20
- 2 entries = 40
- 3 entries = 60
- 4 entries = 80
- 5+ entries = 100

Scores are not judgment. They create visibility.
